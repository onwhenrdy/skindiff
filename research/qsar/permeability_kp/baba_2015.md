# Baba, Tanaka, Sano, Sugiyama, Iwata, Inoue, Yamamoto (2015) — Nonlinear QSPR with SVR/RF

## Citation
Baba, H., Takahara, J., Mamitsuka, H. (2015). In silico predictions of human
skin permeability using nonlinear quantitative structure-property
relationship models. Pharmaceutical Research 32(7):2360-2371.
DOI: 10.1007/s11095-015-1629-y. PubMed: 25735439.

## Equation
Not a closed-form regression. The model uses Support Vector Regression
(SVR, nonlinear RBF kernel) and Random Forest (RF) on a feature set of
molecular descriptors (Mordred / RDKit / DRAGON; the paper details the
selection).

For comparison the paper reports a multiple-linear-regression baseline on
the same 211-compound dataset, which underperforms relative to SVR.

## Training set
n = 211 compounds with permeability coefficients measured under *consistent*
experimental conditions (a key innovation -- the model is fit on a curated
single-protocol subset, not the heterogeneous Flynn aggregate). Aqueous
donor; in-vitro human skin or pig-skin proxy.

## Reported performance
External validation (test set):
- SVR (nonlinear): R^2 = 0.910, RMSE = 0.342, MAE = 0.282 (log Kp units)
- RF: similar, slightly lower
- MLR baseline: substantially worse (R^2 ~ 0.70)

These are the highest test-set performances reported for any Kp QSAR through
2015.

## Validity / limitations
- Black-box: the SVR has no closed-form equation, so users have to deploy
  the trained model (the paper provides supplementary data files / fitted
  model parameters).
- Curated-protocol training means the model is highly accurate *for that
  protocol* but generalisation to the broader Flynn-style aggregate is not
  guaranteed.
- Standard ML-overfit caveats: 211 compounds with hundreds of candidate
  descriptors needs careful cross-validation (the paper does this).

## Notes
Marks the transition from regression QSAR to ML-QSAR for Kp. The headline
finding -- that careful data curation matters as much as model class --
remains relevant: a clean small dataset beats a noisy large one. Subsequent
work by Tsakovska / Asgarkhanova / Kim 2024 / Tippa 2024 builds on this
dataset and methodology.
